# Random Quote Generator

A highly polished, modern, and interactive mobile application built using **Flutter**. This application features a luxury glassmorphic user interface with a fluid dark burgundy gradient background, offering a premium user experience for daily inspiration.

## Key Features
* **Elegant Glassmorphism:** A sophisticated semi-transparent layout card with smooth outer borders and custom glowing text highlights.
* **History Navigation:** A smart background state stack allowing users to effortlessly navigate backward to previously generated quotes step-by-step using the history arrow.
* **Favorites Tray:** A clean bottom-sheet panel that slides up natively, allowing users to aggregate their favorite quotes, view their custom collection, and delete unwanted entries instantly.
* **Responsive Layout:** Completely optimized to render flawlessly on both physical Android devices and high-definition virtual emulators.

## Tech Stack
* **Framework:** Flutter (Multi-platform mobile cross-development)
* **Language:** Dart
* **State Management:** Highly reactive Stateful Widgets utilizing optimization cycles (`setState`).
* **UI Design Paradigm:** Modern Glassmorphic Soft-Ui Core blending premium gradients with subtle semi-transparent overlays.

## Developer Note & Reflection
With this app, I really wanted to focus on high-end visuals and smooth state management. Instead of sticking to basic templates, I challenged myself to build a premium glassmorphic UI—which meant a lot of background gradients, blurs, and layers until the look was perfect.
